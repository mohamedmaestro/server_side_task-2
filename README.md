Calendar Events Management System - Server Side Application

Overview
This is a Node.js/Express server application that provides a RESTful API for managing calendar events. The API supports full CRUD operations for events and includes advanced filtering capabilities to fetch events by day, week, or month. Events can be regular timed events or all-day events, and may span multiple days.

Features
Event Management:

Create, read, update, and delete calendar events

Support for all-day events and multi-day events

Automatic duration calculation (hours and minutes)

Flexible Event Retrieval:

Get all events sorted by start time

Filter events by specific day, week, or month

Handle events that span across date ranges

Database Support:

Works with MySQL

Uses Prisma ORM for database operations
